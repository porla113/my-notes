# Visual Studio Code
## Shortcuts
### General

|Key|Description|
|---|---|
|Ctrl + Shift + P|Show Command Palette.|

### Basic editing

|Key|Description|
|---|---|
|Shift + Alt + Arrow|Copy line up/down.|
|Alt + Arrow|Move line up/down.|
|Ctrl + Shift + Enter|Insert line above.|
|Ctrl + Enter|Insert line below.|

### Useful

|Key|Description|
|---|---|
|Ctrl + K V|Markdown preview side-by-side.|
