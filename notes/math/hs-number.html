Number Theory

ระบบจำนวนจริง

จำนวนจริง

  • \(R\)
  • Real number
  • ทุกจำนวนที่มีตำแหน่งบนเส้นจำนวน
  • ทุกจำนวน ยกเว้น รูทติดลบ และ เลขที่มีส่วนเป็นศูนย์

จำนวนตรรกยะ

  • \(Q\)
  • Rational number
  • เขียนในรูปเศษส่วนได้, ทศนิยมรู้จบ, ทศนิยมไม่รู้จบแต่ซ้ำ
  • \(2 \over 3\), -\(1 \over 4\), 0.1, 3.3333...

จำนวนอตรรกยะ

  • \(Q'\)
  • Irrational number
  • เขียนในรูปเศษส่วนไม่ได้
  • รากที่สองของจำนวนนับ (ที่ถอดค่าออกมาเป็นจำนวนนับไม่ได้) เป็นจำนวนอตรรกยะเสมอ
  • \(\sqrt{2}, \sqrt{3}, \Pi \)

จำนวนเต็ม

  • \(Z\)
  • Integer number
  • จำนวนเต็มบวก, จำนวนเต็มลบ, ศูนย์
  • จำนวนเต็มคู่, จำนวนเต็มคี่
  • ..., -2, -1, 0, 1, 2, ...

จำนวนนับ

  • \(N\)
  • Natural or counting number
  • จำนวนเต็มบวก
  • แบ่งเป็น 1, จำนวนเฉพาะ, จำนวนประกอบ
  • 1, 2, 3, ...

สมบัติของจำนวนจริง

สมบัติปิด

  • นำสมาชิกในเซตใดๆมาดำเนินการกัน แล้วผลลัพธ์ยังเป็นสมาชิกในเซต
  • เซตของจำนวนนับมี สมบัติปิด สำหรับการบวกและการคูณ
  • เซตของจำนวนเต็มและจำนวนตรรกยะมี สมบัติปิด สำหรับการบวก การลบ และการคูณ

เอกลักษณ์

  • จำนวนที่ไปดำเนินการกับจำนวนจริง a ใดก็ตามแล้วได้ผลลัพท์เป็นจำนวน a เดิม
  • \(a * e = e * a = a\)
  • เอกลักษณ์การบวก \(a + 0 = 0 + a = a\)
  • เอกลักษณ์การคูณ \(a \cdot 1 = 1 \cdot a = a\)

อินเวอร์ส หรือ ตัวผกผัน

  • จำนวนที่ไปดำเนินการกับจำนวนจริง a ใดก็ตามแล้วได้ผลลัพท์เป็นเอกลักษณ์
  • \(a * i = i * a = e\)
  • อินเวอร์สการบวก \(a + (-a) = (-a) + a = 0\)
  • อินเวอร์สการคูณ \(a \cdot {1\over a} = {1\over a} \cdot a = 1\)
  • อินเวอร์สการบวกของจำนวนจริงใดๆสามารถหาได้เสมอ
  • อินเวอร์สการคูณของจำนวนจริงใดๆสามารถหาได้เสมอ ยกเว้น 0 ไม่มีจำนวนจริงใดที่คูณกับ 0 แล้วได้ 1

สมการพหุนามและทฤษฏีบทเศษเหลือ

พหุนาม (Polynomial)

  • \(p(x) = a_nx^n + a_{n-1}x^{n-1} + ... + a_1x + a_0\)
  • \(p(x) = 4x^3 - x^2 - 2x + 6\) พหุนามดีกรีสาม
  • สูตรหาคำตอบพหุนามดีกรีสองในกรณีที่แยกตัวประกอบไม่ได้ \(Ax^2 + Bx + C = 0\)
  • \(x = -B\pm \sqrt{B^2 - 4AC \over {2A}}\)
  • ถ้าภายในรู้ทติดลบแสดงว่าพหุนามนั้นแยกตัวประกอบไม่ได้และสมการจะไม่มีคำตอบที่เป็นจำนวนจริง

สูตรแยกแฟคเตอร์ที่ใช้บ่อย

  • \( (x + y)^2 = x^2 + 2xy + y^2\)
  • \( (x - y)^2 = x^2 - 2xy + y^2\)
  • \( x^2 - y^2 = (x + y)(x - y)\) **ใช้ในการ conjugate ด้วย
  • \( x^3 + y^3 = (x + y)(x^2 - xy + y^2)\)
  • \( x^3 - y^3 = (x - y)(x^2 + xy + y^2)\)
  • ตัวอย่าง

ทฤษฏีบทเศษเหลือ (Remainder Theorem)

  • ถ้าหาร \(p(x)\) ด้วย \(x - c\) แล้วจะเหลือเศษเท่ากับ \(p(c)\)
  • เช่น \(p(x) = 4x^2 - 2x + 3\) หารด้วย \(x - 3\) จะเหลือเศษเท่ากับ \(p(3) = 4(3)^2 - 2(3) + 3 = 93\)
  • ถ้าหารลงตัวเหลือเศษเท่ากับ 0 ย่อมกล่าวได้ว่า x - c เป็นตัวประกอบของ p(x)
  • การหารยาว
  • \(x^4 + x^2 - 2x + 3\) หารด้วย \(x^2 + x\)
  • ได้คำตอบคือ \(x^2 - x + 2\) เศษ \(-4x + 3\)
  • หรือสามารถแยกตัวประกอบเป็น \((x^2 + x)(x^2 - x + 2) + (-4x + 3)\)
  • การหารสังเคราะห์ **ใช้ได้เฉพาะตัวหารที่เป็นพหุนามดีกรีหนึ่งเท่านั้น
  • \(x^3 - 3x^2 - 5x + 1\) หารด้วย \(x - 4\)
  • ได้คำตอบคือ \(x^2 + x - 1\) เศษ \(-3\)
  • หรือสามารถแยกตัวประกอบเป็น \((x - 4)(x^2 + x - 1) - 3\)

อสมการพหุนาม

ช่วง (Interval)

  • (2,5) ช่วงเปิด 2 ถึง 5 เซตคำตอบไม่รวม 2 และ 5
  • (2,5] ช่วงเปิด 2 ถึงช่วงปิด 5 เซตคำตอบไม่รวม 2 แต่รวม 5

ข้อควรระวังของอสมการ

  • การบวกหรือลบทั้งสองข้างทำได้เสมอ
  • การคูณและหารทั้งสองข้าง ด้วยจำนวนติดลบต้องพลิกเครื่องหมายของอสมการเสมอ
  • การยกกำลังสอ ถ้าทั้งสองข้างเป็นลบต้องกลับเครื่องหมาย
  • \(a \gt b\) เมื่อ \(a,b \lt 0\) จะได้ \(a^2 \lt b^2\)
  • \(a \gt b\) เมื่อ \(a,b \gt 0\) จะได้ \(a^2 \gt b^2\)

การแก้อสมการ

  • ตัวอย่าง

หลักการที่ต้องรู้

\(y = |x|\)

  • x ∈ R, y ≥ 0
  • x เป็นเลขอะไรก็ได้แต่ y ≥ 0

\(y = \frac{a}{x}\)

  • x ≠ 0

\(y = x^2\)

  • x ∈ R, y ≥ 0
  • x เป็นเลขอะไรก็ได้แต่ y ≥ 0

\(y = \sqrt{x}\)

  • x ≥ 0, y ≥ 0
  • ในรูทห้ามติดลบ ถอดมาก็ห้ามลบ

ค่าสัมบูรณ์ (Absolute)

ทฤษฏีบท

  • \(|x| \ge 0\) เสมอ
  • \(|x| = |-x|\)
  • \(|xy| = |x||y|\)
  • \(|\frac{x}{y}| = \frac{|x|}{|y|}\)
  • \(|x-y| = |y-x|\)
  • \(|x+y| \le |x|+|y|\)
  • \(|x-y| \ge |x|-|y|\)
  • \(|x^n| = |x|^n\) เมื่อ n เป็นจำนวนเต็มใดๆ
  • \(\sqrt{x^2} = |x|\)
  • \(\sqrt{x^2} \neq x\) เพราะ x อาจเป็นจำนวนติดลบได้

หลักในการแก้สมการและอสมการค่าสัมบูรณ์

วิธีที่ 1 simple

  • กำหนดให้ \(a \ge 0\)
  • ถ้า \(|x| = a\) แล้ว \(x = a\) หรือ \(x = -a\)
  • ถ้า \(|x| \lt a\) แล้ว \(-a \lt x \lt a\)
  • ถ้า \(|x| \le a\) แล้ว \(-a \le x \le a\)
  • ถ้า \(|x| \gt a\) แล้ว \(x \lt -a\) หรือ \(x \gt a\)
  • ถ้า \(|x| \ge a\) แล้ว \(x \le -a\) หรือ \(x \ge a\)
  • คิดแบบปกติก็ได้ แปลงเป็นค่าลบให้กลับเครื่องหมายแค่นั้น

วิธีที่ 2 สมมติช่วงเอง

  • \(|x| = x\) ,เมื่อ \(x \ge 0\) เช่น \(|4| = 4\)
  • \(|x| = -x\) ,เมื่อ \(x \le 0\) เช่น \(|-4| = -(-4) = 4\)
