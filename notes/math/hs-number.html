Number Theory

จำนวน

จำนวนจริง

  • \(R\)
  • Real number
  • ทุกจำนวน ยกเว้น รูทติดลบ และ เลขที่มีส่วนเป็นศูนย์

จำนวนเต็ม

  • \(Z\)
  • Integer number
  • ..., -2, -1, 0, 1, 2, ...

จำนวนนับ

  • \(N\)
  • Natural or counting number
  • จำนวนเต็มบวก
  • 1, 2, 3, ...

จำนวนตรรกยะ

  • \(Q\)
  • Rational number
  • เขียนในรูปเศษส่วนได้, ทศนิยมรู้จบ, ทศนิยมไม่รู้จบแต่ซ้ำ
  • \(2 \over 3\), -\(1 \over 4\), 0.1, 3.3333...

จำนวนอตรรกยะ

  • \(Q'\)
  • Irrational number
  • เขียนในรูปเศษส่วนไม่ได้
  • \(\sqrt{2}, \sqrt{3}, \Pi \)

เอกลักษณ์การบวก

  • บวกแล้วได้จำนวนเดิมคือ 0
  • \(a + 0 = 0\)

เอกลักษณ์การคูณ

  • คูณแล้วได้จำนวนเดิมคือ 1
  • \(a \cdot 1 = 1\)

อินเวอร์สการบวก

  • จำนวนที่บวกแล้วได้เอกลักษณ์คือ -a
  • \(-a + a = 0\)

อินเวอร์สการคูณ

  • จำนวนที่คูณแล้วได้เอกลักษณ์คือ 1/a
  • \(1 \over a\)\(\cdot a = 1\) เมื่อ \(a \ne 0\)

สมบัติปิด

  • นำสมาชิกในเซตใดๆมาดำเนินการกัน แล้วผลลัพธ์ยังเป็นสมาชิกในเซต
  • เซตของจำนวนนับมี สมบัติปิด สำหรับการบวกและการคูณ

หลักการที่ควรรู้

y = |x|

  • x ∈ R, y ≥ 0
  • x เป็นเลขอะไรก็ได้แต่ y ≥ 0

y = a ⁄ x

  • x ≠ 0

y = x2

  • x ∈ R, y ≥ 0
  • x เป็นเลขอะไรก็ได้แต่ y ≥ 0

y = √x

  • x ≥ 0, y ≥ 0
  • ในรูทห้ามติดลบ ถอดมาก็ห้ามลบ

ค่าสัมบูรณ์ (Absolute)

ทฤษฏีบท

  • |x| ≥ 0
  • |x| = |-x|
  • |xy| = |x| |y|
  • |x/y| = |x|/|y|
  • |x-y| = |y-x|
  • |x+y| ≤ |x|+|y|
  • |x-y| ≥ |x|-|y|
  • |xn| = |x|n เมื่อ n เป็นจำนวนเต็มใดๆ
  • |x| = √x 2

หลักในการแก้สมการ

วิธีที่ 1 simple

  • กำหนดให้ a ≥ 0
  • ถ้า |x| = a แล้ว x = a หรือ x = -a
  • ถ้า |x| < a แล้ว -a < x < a
  • ถ้า |x| ≤ a แล้ว -a ≤ x ≤ a
  • ถ้า |x| > a แล้ว x < -a หรือ x > a
  • ถ้า |x| ≥ a แล้ว x ≤ -a หรือ x ≥ a
  • คิดแบบปกติก็ได้ แปลงเป็นค่าลบให้กลับเครื่องหมายแค่นั้น

วิธีที่ 2 สมมติช่วงเอง

  • |x| = x ,เมื่อ x ≥ 0 เช่น |4| = 4
  • |x| = -x ,เมื่อ x ≤ 0 เช่น |-4| = -(-4) = 4

วิธีที่ 3 ยกกำลัง 2

  • ตัวอย่าง

สูตรแยกแฟคเตอร์ที่ใช้บ่อย

  • (x + y)2 = x2 + 2xy + y2
  • (x - y)2 = x2 - 2xy + y2
  • (x + y)(x - y) = x2 - y2 **ใช้ในการ conjugate ด้วย
  • (x + y)(x2 - xy + y2) = x3 + y3
  • (x - y)(x2 + xy + y2) = x3 - y3
  • ตัวอย่าง
