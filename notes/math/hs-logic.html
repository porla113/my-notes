สมมูล

  • ประพจน์ที่สมมูลกันคือ ประพจน์สองรูปแบบที่มีค่าความจริงตรงกันทุกกรณี
  • ~(~p)   ≡   p
  • p ∧ q   ≡   q ∧ p
  • p ∨ q   ≡   q ∨ p
  • p ↔ q   ≡   q ↔ p
  • p → q   ไม่มีสมบัติการสลับที่ ***
  • p ∧ (q ∨ r)   ≡   (p ∧ q) ∨ (p ∧ r)
  • p ∨ (q ∧ r)   ≡   (p ∨ q) ∧ (p ∨ r)
  • p → q   ≡   ~p ∨ q   ≡   ~q → ~p   (*** ออกบ่อย)
  • p ↔ q   ≡   (p → q) ∧ (q → p)
  • p ↔ q   ≡   ~p ↔ ~q
  • ~(p ↔ q)   ≡   ~p ↔ q   ≡   p ↔ ~q
  • ~(p ∧ q)   ≡   ~p ∨ ~q
  • ~(p ∨ q)   ≡   ~p ∧ ~q

สัจนิรันดร์

  • รูปแบบของประพจน์ที่มีค่าความจริงเป็นจริงทุกกรณี
  • ให้แทนทั้งก้อนป็นเท็จ ถ้าไม่ขัดแย้ง (เป็นเท็จได้) ไม่เป็นสัจนิรันดร์
  • ให้แทนทั้งก้อนป็นเท็จ ถ้าขัดแย้ง (ไม่ยอมเป็นเท็จ) เป็นสัจนิรันดร์

การอ้างเหตุผล

  • เหตุ (p1, p2, p3, ..., pn) และ ผล (C)
  • (p1 ∧ p2 ∧ p3 ∧ ... ∧ pn) → C
  • สมเหตุสมผล (valid) ก็ต่อเมื่อ แทนเป็นเท็จแล้วขัดแย้ง (เหมือนสัจนิรันดร์)
  • ไม่สมเหตุสมผล (invalid) ก็ต่อเมื่อ แทนเป็นเท็จแล้วไม่ขัดแย้ง (เหมือนสัจนิรันดร์)

ตัวบ่งปริมาณ

ประโยคเปิด

  • ประโยคบอกเล่าหรือปฏิเสธที่มีตัวแปร ซึ่งจะไม่ทราบค่าความจริงจนกว่าจะแทนค่าลงในตัวแปร
  • x + 2 ≥ 9 เป็นประโยคเปิดที่มี x เป็นตัวแปร
  • เขาเป็นติวเตอร์ เป็นประโยคเปิดที่มี เขา เป็นตัวแปร

ตัวบ่งปริมาณ

  • ∀ (for all)
  • ∀x แทน สำหรับ x ทุกตัว
  • ตัวบ่งปริมาณ ∃ (for some)
  • ∃x แทน สำหรับ x บางตัว
  • P(x) แทน ประโยคเปิด และ U เป็นเอกภพสัมพัทธ์
  • ∀x [P(x)] เป็นจริงเมื่อ แทน x ทุกตัวแล้วเป็นจริง
  • ∀x [P(x)] เป็นเท็จเมื่อ แทน x แล้วมีอย่างน้อย 1 ตัวเป็นเท็จ
  • ∃x [P(x)] เป็นจริงเมื่อ แทน x แล้วมีอย่างน้อย 1 ตัวเป็นจริง
  • ∃x [P(x)] เป็นเท็จเมื่อ แทน x ทุกตัวแล้วเป็นเท็จ

สมมูลและนิเสธของของประโยคที่มีตัวบ่งปริมาณ

  • ∀x [P(x)]   ≡   ∀x [Q(x)] ก็ต่อมื่อ P(x)   ≡   Q(x)
  • ∃x [P(x)]   ≡   ∃x [Q(x)] ก็ต่อมื่อ P(x)   ≡   Q(x)
  • ~∀x [P(x)]   ≡   ∃x [~P(x)]
  • ~∃x [P(x)]   ≡   ∀x [~P(x)]
